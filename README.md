# Whiteout Survival Battle simulator
------

This is still a work in progress.
The simulator only supports troop skills up to Tier 7 skills for now.
Feel free to test and give feedback.

# HOW TO USE

1. Edit `assets\fighters.json` with the accounts bonus stats from battle report
2. Edit troops in `main.py`
3. You can change `show_rounds_freq` in `main.py` to show rounds reports
3. run `main.py`
